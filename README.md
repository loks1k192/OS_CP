# OS_CP

Краткое описание
Проект OS_CP — учебная клиент‑серверная система на C. Демонстрирует базовую коммуникацию между клиентом и сервером, простое API сервера и сборку через makefile.

Основные возможности
- Реализация сервера и клиента (TCP/UDP по реализации)  
- Простое серверное API для обработки запросов  
- Makefile для сборки исполняемых файлов  
- Разделение исходников и заголовков в каталоге src/ и include/  
- Документация и примечания в doc/

Требования
- GCC (или совместимый компилятор C)
- make
- Windows (проект разрабатывается в среде Windows; команды и пути соответствуют ОС)

Сборка
1. Открыть терминал в корне проекта:
   cd c:\Users\ASUS\Desktop\labs\CP
2. Выполнить:
   make

Запуск
- После сборки в корне/bin (или согласно Makefile) появятся исполняемые файлы сервера и клиента.
- Запустите сервер:
  ./server
- В другом терминале запустите клиент:
  ./client [опции]

Примеры
- Запустить сервер по умолчанию:
  ./server
- Подключиться клиентом к localhost:
  ./client 127.0.0.1 12345

Структура проекта
- src/ — исходные файлы (.c)  
- include/ — заголовочные файлы (.h)  
- makefile — инструкции сборки  
- doc/ — дополнительная документация  
- .gitignore — файлы, игнорируемые Git

Как вносить изменения
- Вносить правки в src/, обновлять заголовки в include/.  
- Запускать make clean && make для пересборки.
